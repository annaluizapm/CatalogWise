# Catalogwise - Páginas separadas

Agora o protótipo está separado em páginas reais:

- index.html: tela inicial / landing page
- login.html: tela de login
- cadastro.html: tela de cadastro
- principal.html: painel principal depois do login/cadastro
- assets/css/style.css: estilos do projeto
- assets/js/app.js: interações simples

Como testar:
1. Extraia o ZIP.
2. Abra index.html no navegador.
3. Clique em Entrar ou Começar grátis.
4. Ao enviar login ou cadastro, o protótipo vai para principal.html.

Observação:
Este é um protótipo visual. Login e cadastro ainda não salvam dados de verdade.
